# Login-Signup App (Node.js + Express + MongoDB)
Simple authentication system using bcrypt for password hashing and JWT for tokens.

## Setup
1. Clone the repo
2. Run `npm install`
3. Create `.env` file with `JWT_SECRET=your_secret`
4. Run `node app.js`
